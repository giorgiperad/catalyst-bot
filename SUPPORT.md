# Support

CATalyst is beta trading software. For safety, keep public reports free of
wallet secrets, cert files, seed phrases, private keys, and full `.env` content.

## Where to Get Help

- Questions: use GitHub Discussions.
- Confirmed bugs: open a GitHub Issue with the bug report template.
- Security concerns: follow `SECURITY.md` instead of opening a public issue.

## Helpful Bug Report Details

Include:

- CATalyst version
- Operating system
- Sage wallet version
- Pair or asset ticker
- What you expected to happen
- What happened instead
- Short log excerpts with secrets removed

Do not attach live wallet databases, wallet certificates, private keys, seed
phrases, or full unredacted superlogs.
