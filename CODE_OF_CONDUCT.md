# Code of Conduct

CATalyst welcomes careful bug reports, testing notes, ideas, and code review.
This project involves live trading software, so discussions can be stressful.
Please keep them useful and respectful.

## Expected Behavior

- Be specific, kind, and patient.
- Assume good intent while still being clear about risks.
- Keep criticism focused on code, behavior, documentation, or safety.
- Redact secrets before sharing logs or screenshots.
- Help non-developers understand what is happening when possible.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language
- Pressuring anyone to share private wallet material or credentials
- Publicly posting suspected vulnerabilities before maintainers can respond
- Repeatedly derailing issues or discussions after being asked to stop

## Enforcement

Maintainers may edit, hide, lock, or remove comments and may restrict accounts
that make the project unsafe or unproductive for others.

If you need to report a conduct issue, contact the repository maintainer
privately.
